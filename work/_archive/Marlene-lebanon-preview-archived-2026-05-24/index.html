2. The meeting agendas page

The full 2026 schedule is already on the site — every meeting is listed. But most of the Agenda and Minutes slots are blank, so the whole page looks like nobody's been keeping up with it. Swapping those blank cells for "Posted within 7 days" fixes that instantly, and it's actually true — it just sets an expectation.

3. How the site shows up in Google

When residents search "Lebanon Borough" today, there's no description line set on the homepage, so Google improvises with whatever text it finds — currently a random event blurb. Next week it'll change to something else. A single line of clean copy fixes it.

https://www.lebanonboro.com
Lebanon Borough
Tickets & RSVP · Resident Tickets: $15.00 · Non-Resident Tickets: $36.00 · RSVP Deadline: June 30 · Registration: Use the "Register Here" QR code on the flyer.

Google is grabbing a current event blurb because there's no real description set. Next week it could be anything — whatever happens to be on the page that day.

https://www.lebanonboro.com
Lebanon Borough
The official website of Lebanon Borough in Hunterdon County, NJ — council meetings, calendar, departments, forms, and town news.

One line of description text added to the homepage. Next time Google checks the site, it picks up this line and uses it permanently — same description every time, no matter what's on the page that day.
